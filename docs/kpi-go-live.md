# KPI go-live (deckbuilder advisor)

## Adoption
- Taux d'ouverture du panneau Analyse / session deckbuilder.
- % utilisateurs qui consultent le mode expert.

## Actionnabilité
- Taux de decks modifiés après affichage recommandations.
- Nombre moyen de changements de slots / session.

## Qualité perçue
- Taux d'acceptation des recommandations light/premium.
- Feedback "recommandation utile" (thumb up/down).

## Robustesse data
- % recommandations premium émises en mode non-conservateur.
- Taille moyenne d'échantillon matchup lors d'une reco premium.
